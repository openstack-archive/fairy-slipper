
Create tags
===========

.. rest_method::  POST /v2/metadefs/namespaces/tags/{namespace_id}

Creates one or more tag definitions in a namespace.

Error response codes:201,


Request Parameters
------------------

.. rest_parameters:: parameters.yaml

   - namespace: namespace
   - name: name
   - tags: tags
   - namespace_id: namespace_id



Response Parameters
-------------------

.. rest_parameters:: parameters.yaml

   - name: name
   - tags: tags
